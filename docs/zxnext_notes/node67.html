Joystick

To map joystick buttons an $80 is written to nextreg followed by a $00 (left) or $10 (right) to nextreg $29 to select which joystick is being mapped to the keyboard 11 writes to nextgreg $2B to indicate the key(s) for each button. Each byte to addresses the row and column of the ZX Next keyboard with bits 5-3 being the row and bits 2-0 being the column (column 7, 111 is no action).
