Basic Input/Output

The basic I/O (human interface) system of the ZX Spectrum Next supports keyboard, mouse and game controllers. These all extend the functionality found on common ZX Spectrum peripherals. game controllers and PS/2 keyboards can be customized to simulate responses on the stock keyboard.
