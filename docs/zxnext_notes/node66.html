Keyboard

To remap PS/2 keyboard keys, the starting address for the keymap data is written to nextregs $28 and $29 keyboard data address followed by writing the data to nextregs $2A and $2B with the write to $2B autoincrementing the address for the data.

PS/2 keyboard function keys F9-F12 cannot be remapped and have a fixed function.

  • F9 = Multiface NMI/Function key modifier (with 1-8 sends special function F1-F8)
  • F10 = divMMC NMI
  • F11 = expansion bus on
  • F12 = expansion bus off
