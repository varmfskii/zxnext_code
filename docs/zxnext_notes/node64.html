Mouse

A mouse attached to the PS/2 port looks like a kempston mouse to the system. Port $FADF is used to read the state of the mouse wheel and buttons while ports $FBDF and $FBDF are used to read the X and Y positions.

Port $FADF (FADF) Kempston Mouse Buttons

 bits 7-4 = Wheel delta since last read
 bit 3 = fourth button
 bit 2 = middle button
 bit 1 = left button
 bit 0 = right button

Port $FBDF (FBDF) Kempston Mouse X

 bits 7-0 = X coordinate of mouse

Port $FFDF (FFDF) Kempston Mouse Y

 bits 7-0 = Y coordinate of mouse (0-192)
