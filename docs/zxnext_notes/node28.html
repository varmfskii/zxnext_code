Layer 2

Layer 2 is a for bitmapped graphics. It supports modes with resolution at global coordinates (32, 32) to (287, 223) mapped linearly left to right/top to bottom, resolution at global coordinates (0, 0) to (318, 255) mapped top to bottom/left to right, and resolution at global coordinates (0, 0) to (319, 255) with half width pixels mapped so that the nybbles in a byte are adjacent columns (MSN on the left) and bytes running top to bottom/left to right. It can be mapped starting at any 16k memory blocks. The mode requires 3 consecutive blocks (48k) while the others use 5 consecutive blocks (80k).
